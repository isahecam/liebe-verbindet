# 💕 Liebe Verbindet

> *"Love connects"* — An immersive experience for Valentine's Day.

A mobile app that reimagines how people celebrate Valentine's Day using **Virtual Reality (VR)** and **Augmented Reality (AR)** technology — creating shared, memorable moments that go beyond distance.

## Tech Stack

- [Astro 6](https://astro.build) — static site framework
- [Tailwind CSS 4](https://tailwindcss.com) — CSS-first styling via `@tailwindcss/vite`
- [React 18](https://react.dev) — used selectively as Astro islands
- [TypeScript](https://www.typescriptlang.org)

## Commands

| Command           | Action                                          |
| :---------------- | :---------------------------------------------- |
| `npm install`     | Install dependencies                            |
| `npm run dev`     | Start dev server at `localhost:4321`            |
| `npm run build`   | Type-check and build for production (`./dist/`) |
| `npm run preview` | Preview the production build locally            |

## Environment Variables

| Variable  | Description                        |
| :-------- | :--------------------------------- |
| `URL_APK` | Public URL of the downloadable APK |
